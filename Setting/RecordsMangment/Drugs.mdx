title: "Drugs"

### Features

- **Search Drug**: Quickly search for drugs by name.
- **Drug List**: Displays all registered drugs with:
  - **#** – Serial number in the list.
  - **Name** – The drug’s name (e.g., Accupan, Solpadein, Topamax).
- **Actions**:
  - **Edit (✎)** – Modify drug details such as name or dosage.
  - **Delete (⊖)** – Remove a drug from the system.
- **New Drug**: Add a new drug entry to the system.
