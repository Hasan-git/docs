title: "Account"
description: "The Account page allows users to view and manage their account details, tenant information, and application settings."
---

### Sections

1. **User Info**\
   Displays personal details including:
   - Name, Email, Phone
   - Role within the system
   - Account creation date
2. **Tenant Info**\
   Shows information about the tenant (organization/clinic), such as:
   - Tenant name and code
   - Contact and address details
   - Specialty
3. **Application Information**\
   Provides technical details about the application, including:
   - Current build and installed version
   - Option to update the application
   - Local data management (e.g., cleaning cached data)
