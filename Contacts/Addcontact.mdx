## **Adding a New Contact**

- Step-by-step guide:
  1. Go to the Contacts section.
  2. Click **Add Contact**.

     <img
       src="/images/newconatct.png"
       alt="Newconatct Pn"

       className="mr-auto"
       style={{ width:"56%" }}

  3. **Enter basic information: name, Gender, city, phone, email, etc.**
  4. **Click Done.**
  5. **Create patient file if it needed**

## **Fields Included**

- **First name, middle name ,last name**
- **Contact number**
- **Birthdate**
- **Gender**
- **Country**
- **City**
- **Spouse name**
- **Occupation**
- **Email**
